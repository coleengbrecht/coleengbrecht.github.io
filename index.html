Oak Engine

Oak Engine is a game engine I build from the ground up using c++ over the past two years.

Checkout the trailer for an early version here:


The main goal of this engine was to learn as much as possible about what goes into creating a game. I tasked myself with writing all of the core components such as: rendering, physics, asset management, audio mixing, etc...

Features:

  • OpenGL 4 AZDO rendering engine
  • Ray traced shadows
  • Physically based lighting model
  • Bloom post-processing filter
  • HDR rendering pipeline
  • Entity Component System
  • Run time data support with custom type information generator
  • Multichannel audio mixer
  • 2D rigid body physics simulation
  • Builtin scene editor
  • Plugin system
  • Hot code reloading
  • Asset manager
  • Data driven material system
  • Multi distance field font rendering for rendering scalable text from low resolution glyphs
  • Fast AStar pathfinding with path refinement
  • Immediate mode & data oriented gui library
  • Reliable UDP based networking layer

Factured Genome

Fractured Genome (working title) is my latest game project. Is is being build with Oak Engine as a testament to the engines usability and completeness. The game takes place in the far future where an evil cooperation is doing illegal weapons development and you are tasked at stoping them at all costs. More info can be found on the games itch page.
